# Training Center Database Management System

## About

This project was developed as part of the SQL Fundamentals course using MySQL and MySQL Workbench. It simulates a training center database system that manages students, instructors, courses, and enrollments while demonstrating core and advanced SQL concepts.

## Tools

- MySQL
- MySQL Workbench

## Project Features

- Create and manage database tables
- Insert, update, delete, and retrieve data (CRUD)
- Use Primary Key and Foreign Key relationships
- Apply constraints (NOT NULL, UNIQUE, CHECK)
- Use String Functions
- Use Aggregate Functions
- Filter and sort data
- Modify tables using ALTER TABLE
- Perform INNER JOIN, LEFT JOIN, and RIGHT JOIN
- Create and use a VIEW
- Use Window Functions (ROW_NUMBER, RANK, PARTITION BY, LEAD, LAG)

## Database Tables

- Students
- Instructors
- Courses
- Enrollments

## How to Run

1. Open the SQL file in MySQL Workbench.
2. Run the script from the beginning.
3. The database and all tables will be created automatically.
