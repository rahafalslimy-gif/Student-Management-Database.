# Student Management Database

## Overview
This project demonstrates SQL fundamentals using MySQL and MySQL Workbench through a Student Management Database system.

## Database Tables
- Students
- Instructors
- Courses
- Enrollments

## Features Implemented
- CREATE DATABASE and CREATE TABLE
- Primary Keys and Foreign Keys
- INSERT, UPDATE, and DELETE
- Aggregate Functions
- GROUP BY and HAVING
- String Functions
- CASE Statements
- ALTER TABLE
- INNER JOIN, LEFT JOIN, and RIGHT JOIN
- Views
- Window Functions

## Technologies Used
- MySQL
- MySQL Workbench
- SQL
